# 大模型应用开发的工程哲学

当软件系统的核心组件从确定性函数变成概率性语言模型，软件工程的基本原则会发生什么变化，又有什么不会变。

一个软件工程师对"大模型时代的工程方法论"这个问题的系统性回答。内容以个人观点和哲学观为底色，侧重对事物本质结构的揭示和对未来形态的构想。
